# 🚀 Hammad Khan - Professional Laravel Developer Portfolio

A premium, modern, and high-performance developer portfolio built with **React**, **Vite**, and **Tailwind CSS**. This portfolio features a sleek dark-themed SaaS aesthetic, smooth animations using **Framer Motion**, and a dynamic "Technical Arsenal" marquee.

![Portfolio Preview](/public/assets/crm_system.png)

## ✨ Features

- **Premium Dark UI**: Sleek SaaS-style aesthetic with curated glassmorphism and gradient effects.
- **Dynamic Marquee**: An infinite sliding "Technical Arsenal" section showcasing the master tech stack.
- **Smooth Animations**: High-end micro-interactions and scroll animations using Framer Motion and AOS.
- **Project Showcase**: Detailed cards for backend and full-stack projects with live demo and GitHub links.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile to ultra-wide displays.
- **Interactive Experience**: Custom loading screens, scroll progress bars, and floating badges.
*   **Resume Download**: Direct access to my latest CV with a professional download feature.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Interactions**: Framer Motion, AOS (Animate On Scroll)
- **Icons**: React Icons, Lucide React
- **Build Tool**: Vite
- **Typography**: Outfit & Inter (Google Fonts)

## 📁 Project Structure

```bash
├── public/              # Static assets (images, resume, etc.)
├── src/
│   ├── components/      # Reusable UI components
│   ├── data/            # Portfolio data (centralized configuration)
│   ├── pages/           # Main page views
│   ├── styles/          # Global CSS and Tailwind styles
│   └── App.jsx          # Main application logic

```

## 🚀 Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hammadkhan010/hammad_portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd hammad-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
